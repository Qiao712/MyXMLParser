\doxysubsection{Detailed Description}
Interface to a visitor(\+Visitor Pattern). Call a node\textquotesingle{}s accept(\+XMLVisitor\&) with a \mbox{\hyperlink{class_q_simple_x_m_l_parser_1_1_x_m_l_visitor}{XMLVisitor}} object, then the visitor will traverses its every descentant in preorder traversal and call corresponding visit function with the node based on its type.
